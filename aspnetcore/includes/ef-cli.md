如果尚未 `dotnet ef` 安裝，請將其安裝為通用工具：

```dotnetcli
  dotnet tool install --global dotnet-ef
```

如需 EF Core 之 CLI 的詳細資訊，請參閱 [.NET cli EF Core 工具參考](/ef/core/miscellaneous/cli/dotnet)。
