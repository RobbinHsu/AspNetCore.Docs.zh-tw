* `-c|--content`

  提供內嵌 HTTP 要求本文。 例如： `-c "{"id":2,"name":"Cherry"}"` 。

* `-f|--file`

  包含 HTTP 要求本文檔案路徑。 例如： `-f "C:\request.json"` 。

* `--no-body`

  指出不需要任何 HTTP 要求本文。
